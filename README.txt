VISTAAR — Data to Dashboard, Automatically
============================================

HOW TO USE
1. Open vistaar.html in any modern browser (double-click it, no install needed).
2. Upload your Excel/CSV/JSON file (drag-drop or click to browse).
3. If your file has multiple sheets, pick the right one from the dropdown on the review screen.
4. Confirm the auto-detected column types, then click "Build My Dashboard".
5. Use the type-pills (Auto / Target & Collections / Profit & Loss / Sales / Operational / Strategic / Analytical)
   to switch dashboard views, or let the tool auto-pick one based on your column names.
6. Use the 🎨 Theme button (top-right) to switch between 5 color themes: Nebula, Aurora, Sunset, Midnight, Light.
   Your choice is remembered next time you open the file.
7. Export your dashboard as Excel / CSV / JSON / PDF / PNG from the Export button.

WHAT'S NEW IN THIS VERSION
- Smart auto-detection: picks Target & Collections or Profit & Loss view automatically based on
  your column names (Target, Collection/Collxn, Balance, Revenue, Cost, Profit/Loss, etc.)
- Cleans subtotal/grand-total rows and stray section-title rows from Excel exports automatically,
  so totals in your KPIs are correct instead of double/triple-counted.
- Multi-sheet Excel files: auto-picks the richest sheet, with a manual sheet switcher.
- Smarter category-column selection (picks the most useful grouping column, not just the first one).
- 5 selectable color themes with persistent choice.

This is a single self-contained HTML file — no installation, no server, no data leaves your browser.
